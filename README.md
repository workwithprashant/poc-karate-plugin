# POC Karate Plugin

### Karate 0.9.6

Step 1: To build plugin 

`mvn clean install -DkarateVersion=0.9.6`

Step 2: To execute Tests

`mvn clean com.demo:poc-karate-plugin:2.0.0:functionaltest -DkarateVersion=0.9.6`
